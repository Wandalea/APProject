To run the Crystal collection service locally on your computer.

1. First go to your desired GitHub repository and download the files or clone the repository
2. Unzip/open the file in your desired code editor ex. VS Code
3. Make sure you have Python 3 installed. You can check by going to the terminal or command line and typing ```sh python  --version```
4. In the terminal or command line activate the virtual environment using venv\Scripts\activate
5. Install the required packages using ```sh pip install flask flask-session``` or if the application includes a requirements.txt file you will need all of the packages in that. You can use ```sh pip install -r requirements.txt```
6. Then you run the application by typing the command ```sh python app.py``` in the terminal or command line
7. Next you will see a link to a url you can copy and paste it into your browser or hold the control key on you keyboard and click the link
8. Now the web application should show in your browser.
